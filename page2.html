HTML

HTML stands for HyperText Markup Language.

  • HTML is a language.
  • It is commonly used to make a website.
  • HTML is a format that tells a computer how to display a web application.
  • It contain small markup tags.
  • An HTML file must have an html file extension.
  • HTML paragraphs are defined with p tag.
  • HTML links are hyperlinks.
  • In HTML,links are defined with a tag.

CSS

CSS stands for Cascading Style Sheets.

  • CSS is used for styling.
  • CSS saves a lot of work.
  • It can control the layout of multiple web pages at all once.
